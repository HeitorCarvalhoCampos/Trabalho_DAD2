Imagine que você é um mestre construtor de Lego e tem uma pilha de blocos coloridos em sua frente. Agora, em vez de construir uma torre ou um castelo, vamos construir... um site! Mas como? Bem, aqui entra a magia do HTML!

HTML, ou HyperText Markup Language, criado por Tim Berners-Lee na década de 1990, é essencial na formatação de documentos compartilhados online. Com o avanço da internet, tornou-se fundamental na criação de sites, acompanhando a evolução tecnológica ao ser combinado com CSS e JavaScript para desenvolver páginas e aplicativos web dinâmicos.

!DOCTYPE html: O doctype define o tipo do arquivo, no caso html

html lang = “*”: O html lang define a linguagem de leitura, para definir como português do Brasil, basta colocar após o símbolo = entre aspas “pt-br”

head: Aqui é a cabeça do nosso site, onde fazemos a conexão com o JS e CSS, também podemos configurar o titulo e o ícone do nosso site

body: Aqui onde botamos a mão na massa, definimos tudo aqui, como títulos, textos, links, imagens, tabelas, etc... Aqui em geral seria onde nós colocaríamos o conteúdo do nosso site

Ao clicar na caixa de código, você irá para a fase.

        
          <!DOCTYPE html>
          <html lang="en">
          <head>
              <meta charset="UTF-8">
              <meta name="viewport"  content="width=device-width, initial-scale=1.0">
              <title>Document</title>
          </head>
          <body>
              <h1>Titulo</h1>
              <p>Paragrafo</p>
          </body>
          </html>

As Tags são uasadas para informar o navegador qual a estrutura que está sendo construida, o que terá na nossa página. Elas podem indicar parágrafos, título, imagens, links, listas, tabelas etc. As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing).

Tags de texto: são usadas quando quremos por algum texto na tela

<h1>: Essa tag serve para indicar um título. O numero 1 indica a escala de 1 a 6, sendo o 1 o maior título e o 6 o menor.

<p>: Essa tag serve para colocarmos um paragragrafo na nossa página.

Ao clicar na caixa de código, você irá para a fase.

        
          <!DOCTYPE html>
          <html lang="en">
          <head>
              <meta charset="UTF-8">
              <meta name="viewport"  content="width=device-width, initial-scale=1.0">
              <title>Document</title>
          </head>
          <body>
          
            <h1>Titulo</h1>
            <p>Paragrafo</p>
          
          </body>
          </html>

<nav>: Essa tag serve para indicar que estamos fazendo uma seção para navegação.

<a>: Essa tag serve para colocarmos um link.

Ao clicar na caixa de código, você irá para a fase.

        
          <!DOCTYPE html>
          <html lang="en">
          <head>
              <meta charset="UTF-8">
              <meta name="viewport"  content="width=device-width, initial-scale=1.0">
              <title>Document</title>
          </head>
          <body>
          
            <nav>
              <a href="#"></a>
            </nav>
          
          </body>
          </html>

<li>: Indica que estamos colocando um item na lista.

Tags semânticas: Tem o objetivo de descrever o significado do conteúdo presente no documento HTML, tornando mais fácil o entendimento delas.

<header>: Utilizado para representar o cabeçalho de uma página HTML.

<section>: Indica seções ou tópicos da página HTML.

<article>: Utilizamos para declarar um conteúdo que não precisa de outro para fazer sentido.

<nav>: Utilizamos quando precisamos representar um agrupamento de links de navegação.

<aside>: Utilizado quando precisamos criar um conteúdo de apoio/adicional ao conteúdo principal.

<main>: Especifica o conteúdo principal e, consequentemente, de maior relevância dentro da página.

<figure>: É uma marcação de uso específico para a inserção de uma figura.

<footer>: Representa um rodapé de um documento, como a área presente no final de uma página web.

<div>: Utilizado como um contâiner genérico para agrupar blocos de conteúdo.

<span>: Utilizado como um contâiner genérico para agrupar conteúdo em linha.

          <html lang="en">
